PyCoins
=======

Install
------------

Clone the repo, create a python (3.6+) environement and

    make dependencies[-dev]
    make migrate

Run
---

    make runserver

Test
----

    make dependencies-dev
    make test

Manually trigger notification
-----------------------------

    make notify

Shell-plus
----------

    make shell-plus

Full api documentation
----------------------
    make runserver
    go to <server_url>/docs/

Admin
-----
    make runserver
    go to <server_url>/admin/
